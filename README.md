# CISLC-O3

This open-source repository hosts an out-of-order RV64G processor core written in SystemVerilog.

## Goals and Scope

- Implement an out-of-order core that supports RV64G
- Keep the primary RTL in the `rtl/` directory

## Layout

- `rtl/`: SystemVerilog source code
- `run/`: Simulation or build scripts
